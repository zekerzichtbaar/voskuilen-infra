WordPress database merging made easy.

== Description ==

[WPMerge](https://wpmerge.io/ "WordPress database merging made easy") allows users to merge prod and dev DB without losing changes in either of them.

Main features:

*   Dev site changes can be merged with latest prod DB.
*   Download new and modified files as a zip.


== Installation ==

1. Upload the plugin folder to your /wp-content/plugins/ folder
2. Go to the Plugins page and activate WPMerge
